# component-docs

### Folder Structure

```sh
component-docs
 ┣ 📂components     # mdx 컴포넌트
 ┣ 📂metadatas      # scripts로 생성하기 위한 컴포넌트 정보들
 ┣ 📂pages          # 문서
 ┣ 📂public         # 
 ┃ ┗ 📂registry     #
 ┃ ┃ ┗ 📂components # CLI로 install 할 때 참조
 ┣ 📂schemas        #
 ┣ 📂scripts        #
 ┣ 📂seed-design    # snippet은 진실의 원천으로 사용하고, Docs에서 사용하는 코드는 생성해서 사용
 ┗ 📂snippets       # registry에 담김
```
