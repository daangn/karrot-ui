    ### pages/_document.tsx 수정하기

    Seed Design의 컬러 토큰은 `<html />` 태그의 `data-seed`, `data-seed-scale-color` 속성을 통해 라이트/다크모드 전환을 해요.
    Next.js에서 `_document.tsx` 파일을 생성하고 Seed Design의 컬러 토큰을 설정해요.

    <Tabs items={['auto', 'light only', 'dark only']}>
      <Tabs.Tab>
        ```tsx copy filename="pages/_document.tsx" {1,5}
        import { Html, Head, Main, NextScript } from "next/document";

        export default function Document() {
          return (
            <Html data-seed data-seed-scale-color="light" data-seed-scale-letter-spacing="ios">
              <Head>
                <meta name="color-scheme" content="light dark" />
              </Head>
              <body>
                <Main />
                <NextScript />
              </body>
            </Html>
          );
        }
        ```
      </Tabs.Tab>
