# React Headless Libraries

SEED React의 토대가 되는 Headless UI 라이브러리입니다.

SEED 디자인 시스템의 컴포넌트 요구사항을 위한 최소한의 기능을 제공합니다. 일반적인 사용 사례에 적합하지 않을 수 있습니다.
