---
title: Migration to V3
---

Seed Design V3로 마이그레이션하기 위해 필요한 작업들을 안내해요.

<Steps>

<Step>
### 의존성 설치

필요한 의존성들을 설치해요.

```package-install
@seed-design/react
```
</Step>

{/* ----------------------------------------------------------------------- */}
{/* ----------------------------------------------------------------------- */}

<Step>
### CSS import

<Callout>
V2의 `global.css`와 상호운용이 가능하지만, codemod를 통해 V3 스타일시트로 변경하는 것을 권장해요.
</Callout>

```ts title="index.tsx"
import "@seed-design/stylesheet/seed.css";
```
</Step>

{/* ----------------------------------------------------------------------- */}
{/* ----------------------------------------------------------------------- */}

<Step>
### seed-design.json 생성

<Callout>
`seed-design.json` 파일을 생성하여 프로젝트 세팅을 해요.
자세한 설정은 [옵션](/docs/react/get-started/seed-design#옵션)을 참고해주세요.
</Callout>

<Tabs items={['CLI로 추가', '직접 추가']}>
<Tab value="CLI로 추가">
<Steps>
<Step>

### init 명령어 입력하기

</Step>
```sh
npx @seed-design/cli@latest init
```

<Step>
### seed-design 설정
```sh
◇  TypeScript를 사용중이신가요?
│  Yes
│
◇  React Server Components를 사용중이신가요?
│  No
│
◇  seed-design 폴더 경로를 입력해주세요. (기본값은 프로젝트 루트에 생성됩니다.)
│  ./seed-design
```
</Step>
</Steps>
</Tab>

<Tab value="직접 추가">
아래 파일을 프로젝트 루트에 생성해주세요.

```json title="seed-design.json"
{
  "rsc": false,
  "tsx": true,
  "path": "./seed-design"
}
```
</Tab>
</Tabs>

</Step>

{/* ----------------------------------------------------------------------- */}
{/* ----------------------------------------------------------------------- */}

<Step>
### tsconfig.json 설정하기

<Callout>
컴포넌트가 추가되면 `seed-design/ui/{component}`로 import할 수 있도록 `tsconfig.json`에 path를 추가해주세요.
</Callout>

```json title="tsconfig.json"
{
  "compilerOptions": {
    // your options
    "paths": { // [!code highlight]
      "seed-design/*": [ // [!code highlight]
        "./seed-design/*" // [!code highlight]
      ] // [!code highlight]
    } // [!code highlight]
  }
}
```
</Step>

{/* ----------------------------------------------------------------------- */}
{/* ----------------------------------------------------------------------- */}

<Step>
### 필요한 컴포넌트 추가하기

<Callout>
자세한 `add` 명령어 사용법은 [CLI](/docs/react/get-started/cli#add)를 참고해주세요.
</Callout>

```sh
npx @seed-design/cli@latest add action-button
```

이후 프로젝트 루트에서 `./seed-design/ui/{component}`에 컴포넌트가 추가된 것을 확인할 수 있어요.

```tsx title="App.tsx"
import { ActionButton } from "seed-design/ui/action-button";

export default function App() {
  return <ActionButton>라벨</ActionButton>;
}
```

또한 컴포넌트 자세한 `prop`들은 컴포넌트 페이지를 참고해주세요.

</Step>

</Steps>
