### 옵션 테이블

| 속성        | 값                                     | 기본값 |
| ----------- | -------------------------------------- | ------ |
| variant     | outline, normal, info, warning, danger | normal |
| title       | text                                   |        |
| description | text                                   |        |
| icon        | Icon                                   |        |
