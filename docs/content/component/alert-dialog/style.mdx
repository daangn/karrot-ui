3. Text Container

7. Primary Button - Button Primitive

## Visual Options

| Property    | Value                                                          | Description           |
| ----------- | -------------------------------------------------------------- | --------------------- |
| variant     | single, vertical, horizontal-secondary, horizontal-primary-low | 버튼 위계에 대한 옵션 |
| title       | text                                                           | 타이틀에 대한 옵션    |
| description | text                                                           | 설명에 대한 옵션      |
