## 구조도

1. Backdrop - Alert Dialog Primitive (Backdrop)
2. Container - Alert Dialog Primitive (Container)
3. Text Container
4. Title - Alert Dialog Primitive (Title)
5. Description - Alert Dialog Primitive (Description)
6. Action Container
7. Main Action - Button Primitive
8. Main Action Label - Button Primitive
9. Sub Action - Button Primitive
10. Sub Action Label - Button Primitive

## Visual Options

| Property           | Value                                 | Description           |
| ------------------ | ------------------------------------- | --------------------- |
| action align       | single, vertical, horizontal          | 버튼 정렬에 대한 옵션 |
| title              | text                                  | 타이틀에 대한 옵션    |
| description        | text                                  | 설명에 대한 옵션      |
| sub action variant | primary-low, secondary, secondary-low | Sub Action 위계       |
