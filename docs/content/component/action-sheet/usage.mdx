### 옵션 테이블 (Action Sheet)

| 속성        | 값          | 기본값 | 설명 |
| ----------- | ----------- | ------ | ---- |
| is open     | false, true | false  |      |
| title       | text        |        |      |
| description | text        |        |      |

### 옵션 테이블 (Action)

| 속성    | 값                   | 기본값  | 설명 |
| ------- | -------------------- | ------- | ---- |
| label   | text                 |         |      |
| variant | default, destructive | default |      |
