### 옵션 테이블

| 속성        | 값          | 기본값 | 설명                                                 |
| ----------- | ----------- | ------ | ---------------------------------------------------- |
| is disabled | true, false | false  | True일 경우, Range Slider가 유저와 상호작용하지 않음 |
| label       | text        |        |                                                      |
| min         | number      | 0      |                                                      |
| max         | number      | 100    |                                                      |
| step        | number      | 1      |                                                      |
