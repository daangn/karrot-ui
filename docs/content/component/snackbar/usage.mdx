| 속성               | 값                        | 기본값  | 설명 |
| ------------------ | ------------------------- | ------- | ---- |
| type               | default, success, warning | default |      |
| message            | text                      |         |      |
| action label       | text                      |         |      |
| pauseOnInteraction | false, true               | true    |      |

<br />

<DoDontLayout>
  <DoBox>

  </DoBox>
  <DontBox>
    <DontImage>
      ![snackbar guideline](./images/snackbar-guide-dont2.png)

      제거하며, 핵심 메세지만 전달합니다.
    </DontText>
  </DontBox>
